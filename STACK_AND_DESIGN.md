# Стек технологий и Дизайн-система сайта

Этот документ подробно описывает стек технологий, архитектуру, интерактивные возможности и дизайн-систему портфолио Якова Пилипюка ([yapil.art](https://yapil.art)).

---

## 🛠️ Технологический Стек (Tech Stack)

Проект построен с использованием современных фронтенд-технологий, ориентированных на производительность, типографику и продвинутые визуальные эффекты.

*   **Фреймворк:** **Next.js (v16.2.9)** с архитектурой **App Router**. Поддерживает серверные компоненты (React Server Components), быструю гидратацию и оптимизацию шрифтов/ресурсов.
*   **Библиотека рендеринга:** **React (v19.2.4)**.
*   **Язык программирования:** **TypeScript (v5.x)** для строгой типизации и безопасности.
*   **Стилизация:**
    *   **Tailwind CSS (v4)** и `@tailwindcss/postcss` для утилитарных стилей.
    *   **Vanilla CSS Variables** (в `globals.css`) для управления темами и токенами дизайна.
    *   `clsx` и `tailwind-merge` для гибкого динамического управления классами.
*   **Анимации и Переходы:**
    *   **Framer Motion (v12 / motion/react)** для 3D-переходов между страницами, скролл-анимаций и микро-интерактива.
    *   **Lenis (v1.3.23)** для плавного скроллинга (Smooth Scroll).
*   **WebGL и Графика:**
    *   **Three.js (v0.184.0)** для поддержки сложной интерактивной графики.
    *   **HTML5 Canvas (2D & WebGL)** для рендеринга интерактивных фонов и эффектов в реальном времени.
    *   `simplex-noise` для генерации процедурного шума.
*   **Вспомогательные инструменты:**
    *   `lucide-react` для легковесных векторных иконок.
    *   `react-medium-image-zoom` для интерактивного увеличения изображений в кейсах.

## 📐 Концепция Дизайна и Дизайн-система

Визуальный характер сайта вдохновлен философией **экстремального текстового минимализма** и напоминает чистую, строгую статью в Notion или профессиональный журнальный лонгрид. 90% контента — это выверенная типографика и микро-анимации.

### 1. Типографика
*   **Основной шрифт:** **Inter** (подключен из Google Fonts).
*   **Настройки:** Плотный межстрочный интервал (`line-height: 1.68`) и отрицательный трекинг (`letter-spacing: -0.02em` или `tracking-tight`), создающие ощущение плотного журнального набора.
*   **Иерархия размеров:**
    *   *Главный заголовок / Имя:* Не более `24px` / `text-2xl` с насыщенностью `semibold` (хотя элемент `.site-name` в шапке масштабируется до `72px` с трекингом `-0.08em` для эффекта постера на первом экране, в остальных местах сохраняется скромность).
    *   *Названия секций:* `10.5px` (капсом, с большим межбуквенным интервалом `letter-spacing: 0.14em`, цвет `var(--fg-muted)`).
    *   *Основной текст:* Стандартный журнальный размер `15px` (`text-base`).

### 2. Цветовая палитра и темы
Сайт поддерживает бесшовное переключение тем (Light/Dark) без вспышек цвета при перезагрузке (с помощью инлайн-скрипта в `layout.tsx`, читающего `localStorage`).

| Элемент / Токен | Дневная тема (По умолчанию) | Ночная тема (Dark) |
| :--- | :--- | :--- |
| **Общий фон сайта (`--bg`)** | `#E8EFF2` (мягкий серо-голубой) | `#000000` (глубокий черный) |
| **Центральный шейп (`--surface`)** | `#E8EFF2` | `#161B1E` (темно-серый) |
| **Основной текст (`--fg`)** | `#000000` | `#FFFFFF` |
| **Приглушенный текст (`--fg-muted`)**| `rgba(0, 0, 0, 0.42)` | `rgba(255, 255, 255, 0.38)` |
| **Акцент / Интерактив (`--accent`)**| `#FD4B32` (неоновый кораллово-красный)| `#FD4B32` |
| **Границы (`--border`)** | `rgba(0, 0, 0, 0.1)` | `rgba(255, 255, 255, 0.1)` |

### 3. Сетка (Grid) и Лэйаут
*   **Сетчатый контейнер (Notion-style):** Весь контент выровнен по центру в строгую вертикальную колонку. Ширина колонки (`.content-col`) составляет `50%` на десктопах (до `40%` на ультрашироких экранах) и `100% - 16px` на мобильных устройствах.
*   **Аккуратные разделители:** Логические секции разделяются тонкими 1-пиксельными линиями (`hr.divider`), что сохраняет концепцию чистого листа.

## ✨ Ключевые интерактивные элементы и фичи

1.  **Интерактивный ASCII-портрет ([AsciiPhoto.tsx](file:///Users/yapil/Documents/0/yapil/src/components/AsciiPhoto.tsx)):**
    *   Располагается на первом экране.
    *   В реальном времени сканирует пиксели изображения `/me.webp` и отрисовывает их на `canvas` в зависимости от яркости: темные участки преобразуются в плотные штрихи, круги и квадраты, светлые — в ASCII-символы, точки или остаются прозрачными.
    *   Добавлен "глич-эффект": случайные пиксели средней яркости раскрашиваются в случайные RGB-цвета.
    *   Между темными точками лица динамически строятся и плавно колеблются цветные кривые Безье, концы которых подсвечиваются яркими RGB-точками.
2.  **Задний анимированный фон ([falling-pattern.tsx](file:///Users/yapil/Documents/0/yapil/src/components/ui/falling-pattern.tsx)):**
    *   Слой под контентом (`-z-10`), создающий ощущение падающего цифрового дождя из тонких штрихов и точек разной степени прозрачности.
    *   Поверх анимации наложена сетка из мелких точек с помощью радиального градиента `radial-gradient`.
3.  **Кубический 3D-переход между страницами ([ThreeDPageTransition.tsx](file:///Users/yapil/Documents/0/yapil/src/components/ThreeDPageTransition.tsx)):**
    *   Кастомный переход на базе `AnimatePresence` из Framer Motion.
    *   При смене маршрута (например, переход на страницу кейса) текущая страница плавно уплывает по оси Y (`rotateY` до `-90` или `90` градусов в зависимости от направления) и отдаляется по оси Z, а новая страница вращается ей навстречу.
    *   Код отслеживает текущее положение скролла (`window.scrollY`) при переходе, чтобы уходящая страница плавно исчезала из своей позиции, исключая резкие скачки. На время анимации скролл Lenis блокируется.
4.  **Следящее превью кейсов при наведении ([Cases.tsx](file:///Users/yapil/Documents/0/yapil/src/components/Cases.tsx)):**
    *   На главной странице в списке кейсов при наведении на элемент с помощью портала (`React Portal`) на весь экран рендерится превью (картинка `.webp` или зацикленное видео `.webm`/`.mp4`).
    *   Превью плавно следует за курсором с использованием алгоритма линейной интерполяции (Lerp) со сглаживанием `0.08`, а также слегка поворачивается при быстром движении и увеличивается.
5.  **Плавный скролл ([SmoothScroll.tsx](file:///Users/yapil/Documents/0/yapil/src/components/SmoothScroll.tsx)):**
    *   Инициализация библиотеки **Lenis** для обеспечения кинетического скролла на всех устройствах.
6.  **Текстовые эффекты:**
    *   **MagicText ([MagicText.tsx](file:///Users/yapil/Documents/0/yapil/src/components/MagicText.tsx)):** Абзацы текста, слова в которых постепенно подсвечиваются от полупрозрачного состояния (`opacity: 0.2`) до полного белого/черного по мере прокрутки страницы пользователем.
    *   **HighlightHover ([HighlightHover.tsx](file:///Users/yapil/Documents/0/yapil/src/components/HighlightHover.tsx)):** При наведении на ссылки текст подчеркивается линией, которая плавно увеличивается по высоте на 100%, превращаясь в фоновую плашку акцентного цвета, а цвет букв меняется на контрастный белый с помощью сдвига маски.
7.  **Локальное время ([Clock.tsx](file:///Users/yapil/Documents/0/yapil/src/components/Clock.tsx)):**
    *   В шапке отображается точное текущее время в формате `ЧЧ:ММ:СС` с жесткой привязкой к часовому поясу **UTC+5** (Казахстан).
